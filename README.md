# Таймлист

Система учёта рабочего времени и ИИ-анализа созвонов (аналог 1С Таймлист).

## Возможности

- **Сотрудники** — база сотрудников, должности, отделы
- **Табель** — приход/уход, подсчёт часов, переработки
- **Отсутствия** — отпуска, больничные, командировки, отгулы, удалёнка
- **Созвоны и конференции** — загрузка транскриптов, ИИ-анализ через Ollama
- **Календарь** — визуализация отсутствий и созвонов
- **Дашборд** — сводная статистика

## ИИ-анализ созвонов (Ollama)

Анализ транскриптов созвонов с помощью локальной LLM (Ollama):
- Резюме встречи
- Извлечение задач и ответственных
- Ключевые темы обсуждения
- Принятые решения
- Определение тональности (позитивная/нейтральная/негативная)

## Быстрый старт

### 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 2. Установка Ollama

```bash
# Linux
curl -fsSL https://ollama.com/install.sh | sh
ollama pull llama3
```

### 3. Запуск

```bash
python -m app.main
# или
uvicorn app.main:app --reload
```

Откройте http://localhost:8000

### Docker

```bash
docker-compose up -d
```

## Переменные окружения

| Переменная | По умолчанию | Описание |
|---|---|---|
| `DATABASE_URL` | `sqlite:///./timelist.db` | Строка подключения к БД |
| `OLLAMA_BASE_URL` | `http://localhost:11434` | URL Ollama API |
| `OLLAMA_MODEL` | `llama3` | Модель для анализа |

## API

Документация API доступна по адресу: http://localhost:8000/docs

## Технологии

- **Backend:** Python, FastAPI, SQLAlchemy, SQLite
- **Frontend:** HTML, CSS, JavaScript (без фреймворков)
- **AI:** Ollama (локальная LLM)
